Базовые операции по DOM

  • Навигация по DOM

    Доступ к DOM начинается с объекта document, из него можно добраться к любым элементам.

  • Поиск элементов

    Группа методов elem.querySelector* – это современный стандарт для поиска элементов.

  • Свойства и атрибуты

    elem.textContent, elem.classList, elem.style, elem.hasAttribute(name), elem.getAttribute(name), elem.setAttribute(name, value), elem.removeAttribute(name)

  • Создание и удаление элементов

    DOM API позволяет не только выбирать или изменять существующие, но и удалять, а также создавать новые элементы, после чего добавлять их в документ.
